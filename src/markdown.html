SPSSINC TOBIT REGR Extension Command

Calculate a Tobit regression model

SPSSINC TOBIT REGR DEPENDENT=dependent variable*
ENTER=independent variables* DISTRIBUTION=GAUSSIAN** or EXPONENTIAL or LOGISTIC or LOGNORMAL or WEIBULL or LOGLOGISTIC
LOWERBOUND=number UPPERBOUND=number

/OPTIONS MISSING=LISTWISE** or FAIL EXECUTE=TRUE** or FALSE

/SAVE COEFSDATASET=datasetname PROGRAMFILE=“filespec*”

* Required
* Default

SPSSINC TOBIT REGR /HELP prints this information and does nothing else.

SPSSINC TOBIT REGR DEPENDENT=mpg ENTER=engine weight
LOWERBOUND=0.

Split files and weight are not honored by this command.

DEPENDENT and ENTER specify the dependent and independent variable names. Categorical independent variables are automatically converted appropriately to factors. A constant term is automatically included.

LOWERBOUND and/or UPPERBOUND specify the lower and/or upper limits for the dependent variable. If neither is used, there are no limits applied, and the method amounts to ordinary regression.

DISTRIBUTION specifies the error distribution. The choices are

  • GAUSSIAN
  • EXPONENTIAL
  • LOGISTIC
  • LOGNORMAL
  • WEIBULL
  • LOGLOGISTIC

OPTIONS

MISSING=LISTWISE causes listwise deletion of missing values.
FAIL stops the procedure if missing values are encountered.

EXECUTE=FALSE runs the command syntax without running the Tobit regression.
This is mainly useful in combination with SAVE PROGRAMFILE.

SAVE

COEFSDATASET causes a dataset containing the coefficients and the error scale parameter to be created. The dataset name must not already be in use.

PROGRAMFILE causes the R code that implements the tobit regression to be written to the specified file. Since the tobit function has features not exposed in this extension command, the generated program can be a useful starting point for additional specifications.
